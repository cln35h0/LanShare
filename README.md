# LanShare

A local LAN file sharing server built with Flask. Upload files from your PC or phone using drag and drop, track progress and transfer speed live, and share instantly over your local network.

## Features

- Drag and drop file uploads
- Live upload progress bar
- Transfer speed and ETA
- QR code for phone access
- View uploaded files in browser
- Open files directly
- Download files
- Delete files
- Works on LAN

## Requirements

Install dependencies:

```
pip install flask qrcode pillow werkzeug
```

## Installation

Clone the repository:

```bash
gh repo clone cln35h0/LanShare
cd LanShare
```

To run:
```
python LanShare.py
```
